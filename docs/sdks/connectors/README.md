# Connectors
(*Connectors*)

## Overview

Retrieve metadata for connectors.

### Available Operations

* [ListMeta](#listmeta) - List Connectors Meta Information for all providers
* [GetMeta](#getmeta) - Get Connector Meta information for the given provider key
